This is a project carried out by COMPUTER SCIENCE:GROUP B AI GROUP 5, it employs the use of a softbot which in this case would act as a tour guide, using Nigeria as a case study (place of tour). It will aid users in a "virtual tour", it will also provide answers to some users questions, based on the tour that has been given
